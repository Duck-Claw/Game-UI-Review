Install

下载后放入 Codex Skills 目录

Skill 文件夹名是 game-ui-review。安装后重启 Codex，即可使用 $game-ui-review 调用。

bash — install 
mkdir -p ~/.codex/skills
cp -R game-ui-review ~/.codex/skills/game-ui-review

# 重启 Codex 后调用：
$game-ui-review 欧美 SLG 活动页，上线前检查。
