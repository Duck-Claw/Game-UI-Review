Review Coverage

一次评审覆盖关键风险

01

层级与可读性

判断主行动、重点信息、小字、数值和复杂背景上的阅读压力。

02

市场适配

针对欧美、日韩、东南亚与全球市场识别风格、密度和转化表达风险。

03

一致性检查

检查按钮、图标、材质、描边、阴影、发光和组件化规则。

04

落地风险

从 Unity 还原、切图、混合模式、安全区、多比例和多语言空间评估实现成本。

Install

下载后放入 Codex Skills 目录

Skill 文件夹名是 game-ui-review。安装后重启 Codex，即可使用 $game-ui-review 调用。 

mkdir -p ~/.codex/skills
cp -R game-ui-review ~/.codex/skills/game-ui-review

# Restart Codex, then invoke:
$game-ui-review 欧美 SLG 活动页，上线前检查。
